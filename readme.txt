Heartbleed Check Floating Tab

== Description ==

A simple floating tab displaying the Heartbleed Bug logo with a link to an external checker confirming that your site is free from the bug
== Installation ==

1. Under the plugin menu choose "Add new"
2. Choose Upload
3. Choose heartbleed-check.zip and hit "Install Now"
4. Activate the plugin
